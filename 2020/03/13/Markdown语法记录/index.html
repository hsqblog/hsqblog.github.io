Markdown

归属： 轻量级标记语言

创始人： 约翰·格鲁伯（英语：John Gruber）

创建时间： 公元纪年2004年

文档后缀： .md .markdown

应用： Gitbook、Blog……

编辑器： Typora、VSCode……

语法详解

标题

使用#号可以表示1-6级标题，#号的数量对应标题的级数，级数越小字体越小，层级越小


 # 一级标题
 ## 二级标题
 ### 三级标题
 #### 四级标题
 ##### 五级标题
 ###### 六级标题

段落

Markdown 的段落没有特殊格式，这里主要介绍一下段落换行：

  • 使用段落末尾加两个以上空格 + 回车
  • 使用空行代表换行

字体


*斜体*
_斜体_
**粗体**
__粗体__
***粗斜体***
___粗斜体___

线条

  • 分割线

***
* * *
*****
- - -
-------
  • 删除线

~~删除线条~~
  • 下划线

<u>下划线</u>    <!--使用HTML语法中的<u>标签代替-->

脚注


[^脚注]
[^脚注]: 脚注内容

列表

  • 无序列表

    使用*号 +号 -号都可标记无序列表


* 无序列表
+ 无序列表
- 无序列表
  • 有序列表

1. 有序列表1
2. 有序列表2
3. 有序列表3
  • 列表嵌套

1. 有序列表
    - 嵌套列表     <!--在嵌套列表前面加上四个空格实现嵌套-->

引用

在段落前使用> 号实现区块引用


> 区块引用
>> 区块嵌套一层
>>> 区块嵌套两层

代码

使用两个 ` 号将段落中的代码引起来，实现代码显示


使用`printf()`函数     <!--使用代码-->

代码块

  • 使用四个空格或Tab键
  • 使用两个```扩起来并在后面制定语言，达成代码高亮

    #include <stdio.h>
    int main(){
            printf("REDX");
            return 0;
    }

```php    <!--指定代码语言-->
    <? php
        echo "REDX";
    ?>
```

[链接名称](链接地址)

<链接地址>
