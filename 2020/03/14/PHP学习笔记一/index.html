介绍

PHP 是一门使服务器产生动态输出的编程语言，输出的结果根据每次浏览器请求页面响应的结果而不同。

PHP基本结构


<?php
  代码语句;
  ……
?>
  • PHP 所有的代码都要在开始标记<?php符号和结束标记?>之内，PHP解释器以发现<?php开始解释，遇到?>结束解释。
  • PHP 所有的语句以;分号作为结束标示，没有分号结尾的语句会报错的。
